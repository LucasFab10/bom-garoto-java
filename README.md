GoodDog 🐶

Projeto desenvolvido em Java para praticar conceitos básicos de Programação Orientada a Objetos (POO).

Objetivo

Criar diferentes objetos da classe `GoodDog` com tamanhos distintos e fazer com que cada um apresente um comportamento diferente ao executar o método `bark()`.

Conceitos praticados

- Criação de classes e objetos
- Encapsulamento (`private`)
- Métodos getters e setters
- Variáveis de instância
- Estruturas condicionais (`if`, `else if`, `else`)
- Tomada de decisão baseada em atributos do objeto

Funcionamento

Cada cachorro recebe um tamanho através do método `setSize()`.

Dependendo do tamanho informado, o método `bark()` produz uma saída diferente:

- Acima de 60 → `Woof! Woof!`
- Entre 15 e 60 → `Ruff! Ruff!`
- Abaixo de 15 → `Yip! Yip!`
